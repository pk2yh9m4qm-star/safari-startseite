# Safari-Startseite einrichten

1. Öffne `index.html` in Safari.
2. Kopiere die Adresse aus der Safari-Adressleiste. Sie beginnt mit `file:///`.
3. Öffne Safari > Einstellungen > Allgemein.
4. Setze "Neue Fenster öffnen mit" auf "Homepage".
5. Füge die `file:///`-Adresse bei "Homepage" ein.

Optional kannst du auch "Neue Tabs öffnen mit" auf "Homepage" setzen.

## Nutzung

Wähle oberhalb der Suche Google, ChatGPT oder Claude. Die runde Auswahlfläche gleitet zum aktiven Dienst; deine Auswahl bleibt lokal gespeichert.

Mit dem Plus-Knopf kannst du eine Bild- oder Datei auswählen. Der Dateiname wird danach unter der Suche angezeigt. Für eine Suche den Text eingeben und die Eingabetaste drücken.
